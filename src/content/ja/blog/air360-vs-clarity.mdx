## まとめ : なぜ Air360 を選ぶのか？

- **完全なデータ所有権**： Air360では、訪問者データはお客様のものであり、お客様だけのものです。Clarityが広告やAIトレーニングを通じて訪問者データを収益化する一方で、Air360を使用してサードパーティのデータ送信はありません。
- **Cookieless 追跡とデータ処理の最小化**： Air360は、強力なインサイトを提供しながら、最小化を含む個人情報保護法の遵守を保証します。Air360 では、顧客は顧客であり、当社の製品ではありません。私たちの顧客は、私たちの最優先事項である顧客の利益を保証するサービスにお金を支払います。

## Microsoft Clarityとは何ですか？

最近、Microsoft Clarity が無料の分析ツールとして市場に登場しました。

Microsoft Clarityは、セッションのレコーディング、ヒートマップ、行動インサイトをすべて無料で提供します。しかし、隠れたコストはデータの所有権とデータの共有にあります。収集されたデータは、Microsoft が広告ターゲティングを最適化するために使用し、他の場所で訪問者に表示される広告に影響を与える可能性があります。

マイクロソフトは、Clarity を通じて収集されたこのデータが、マイクロソフトの広告ネットワークおよび AI モデルを向上させるために使用されることに同意します。つまり、お客様の Web サイトでのユーザーの対話は、マイクロソフトの広告機能を強化するために活用される可能性があります。Clarity は、訪問者の行動データを追跡し、マイクロソフトに送信します。マイクロソフトは、お客様の訪問者および顧客のデータをマイクロソフト独自の目的で使用する権利を留保します。

たとえデータが匿名化されていても、行動データは分析だけとは異なる目的で価値があります。潜在的には、これらのデータはマイクロソフトによって競合他社の利益のために使用される可能性があります。

いずれにせよ、マイクロソフトがユーザーデータを使用または共有する方法が、ユーザーの同意や合意された目的を含むGDPR/データプライバシー法の要件に準拠しているかどうかを深く評価する必要があります。


## Microsoft Clarityは本当に無料ですか？

諺にもあるように、**無料の昼食**はありません。

Clarityのようなツールが「無料」として販売されている場合、上記で述べた懸念の他にも、その採用には隠れたコストが発生します。

初期費用として金銭的な出費はないかもしれませんが、このようなツールを効果的に運用するには、多くの場合、時間と労力の多大な投資が必要です。このパターンは今に始まったことではなく、15年近く前にGoogle Analytics（GA）が同様の前提で市場に参入した。紙の上では、GAは無料でしたが、多くの企業はすぐに、自社の特定のニーズに対応させるために、プラットフォームの設定、管理、解釈に特化したチーム全体が必要であることに気づきました。

同様に、Clarityは一見ユーザーフレンドリーに見えますが、深く掘り下げると、意味のある洞察を得るにはかなりの手作業が必要であることがわかります。実際、通行料を比較するということは、従業員が設定を微調整したり、実用的なデータを抽出しようとして費やした時間を計算することを意味します。そうすると、本当のコストが見えてくる。皮肉なことに、歴史は繰り返される。プレゼントのように見えるものが、予想以上に高価なものになってしまうのだ。
その代わりに、その設計はプロバイダーの目標に応えるものである-彼らの利益のためにユーザーに関する貴重なデータを収集する。

Clarityを使えば、企業は無料のツールを手に入れることができるかもしれないが、その代わりに、より少ないサービスに対して、より大きなコストが発生する可能性がある。

## ClarityよりもAir360を選ぶ6つの理由
Microsoft Clarityがどのようなもので、なぜ無料のツールとして紹介されているのか、お分かりいただけたと思います。ウェブサイトのデータについて心配になってきたなら、Air360 のようなプライバシーを第一に考えた分析を検討し始めることができます。
Air360がClarityと比較して優れている6つの主な違いは以下の通りです。

### 1. プライバシーを第一に考える Clarity に代わるもの

Air360 では、アナリティクスはサードパーティの広告ネットワークではなく、クライアントのビジネスに役立つべきだと考えています。そのため、プライバシーを中核に置いたプラットフォームを構築しています。マイクロソフトのClarityとは異なり、Air360 の最新リリースは完全にクックレスであり、ユーザー追跡が個人のプライバシーを損なうことはありません（そして、より多くのユーザーを追跡します！）。

さらに重要なことは、収集されたデータ（自動的に仮名化されます）が第三者に販売、共有、送信されることはありません。

### 2. あなたとチーム

Air360 と Microsoft Clarity のもう一つの大きな違いは、お客様が受けるサポートと専門知識のレベルです。Air360 は単なるツールではなく、真のビジネス価値を引き出すための専門家チームによる完全なソリューションです。Clarityを使用する場合、データの意味とそれに対する対処方法を理解するのはあなた自身です。また、前述のように多大なコストがかかります。

さらに、「知らないことによる実際のコストはいくらか」という問題もあります。解約の理由、ファネルの漏れ、コンバージョンの低下など、重要な洞察を見逃すことによる機会コストは、莫大なものになる可能性がある。適切なガイダンスがなければ、企業は間違った問題を追うために貴重な時間とリソースを浪費するかもしれません。

Air360 では、異なるアプローチを取ります。私たちの顧客は、単なるデータ以上のものを得ることができます-彼らは、戦略的な洞察力と、それらの洞察を有意義な行動に変えるための専門家のガイダンスを得ることができます。目標は、データを理解し、ウェブサイトを改善することです。Air360 では、お客様の成長の可能性を最大化するために、お客様と共に働くチームがあります。これは、長年にわたって幸せで長期的な顧客を維持するための私たちの秘密のレシピの一つです。

### 3. 成長を妨げる機能格差

機能面では、Clarityはセッションのレコーディングや基本的なヒートマップなど、初心者や予算があまりない組織にとって魅力的な機能をいくつか提供しています。

しかし、これらの機能は、コンバージョン率の実質的な改善を促進するために必要な機能の表面をほとんどかすっていません。

例えば、ファネル分析は、購入やサインアップのような重要なプロセスにおいて、ユーザーがどこで脱落したかを理解するための重要な要素です。この機能がないと、企業は特定のページがコンバージョンに至らない理由を推測することになります。

さらに、Clarityには分析するデータセットに関する透明性がないため、どのような情報を基に意思決定を行っているかを正確に把握することが困難です。
ユーザー・ジャーニー・マッピングも、このツールキットにはない重要な機能です。訪問者がウェブサイトをどのようにナビゲートするかを理解することは、摩擦点や最適化の機会を特定するのに役立ちます。このような重要な機能を備えていないため、Clarityはユーザーに包括的なデータを提供するのではなく、断片的な洞察をまとめるのに苦労させることになります。使い始めたばかりのユーザーや、データ駆動型の意思決定を優先しないユーザーにとっては機能的ですが、「何もしないよりはマシ」と言うのは、かすかな賞賛で非難しているように感じられます。

### 4. データの質： 精度が重要な理由

マイクロソフトのClarityは、ウェブサイト上で起きていることの部分的なビューを提供します。
その結果は？サイトのコンバージョン率を真に最適化するために必要な深い理解よりも、表面レベルの洞察を得ることができます。

一方、Air360 はコンバージョン率の問題を発見することを目的としています。Air360 は、基本的なセッションのレコーディングやヒートマップをはるかに超えています。完全なユーザー・ジャーニー分析、詳細な行動追跡、高度なファネル分析を提供し、ユーザーがどこで、なぜ離脱したかを正確に突き止めるお手伝いをします。

その製品ページの特定のCTAにエンゲージするのにかかる時間は？特定のA/BテストがこのCTAと貴社の収益にどのような影響を与えるのか？ユーザーはためらっているのか？言語、デバイス、セグメントはこのCTAにどのような影響を与えるのか？これらは、Clarityがお手伝いできない重要な質問です。私たちは、これらが重要なビジネス上の意思決定を行うために重要であると考えています。

Air360には、意味のある分析を得るためにユーザーをセグメント化する強力かつシンプルな方法があります。カスタムユーザーセグメントを作成することも、正しい分析において大きな役割を果たします。新規の訪問者とリピーターの訪問者、または異なる国からの訪問者をセグメント化しないと、せいぜい何もできないインサイトになります。Air360 は、セグメンテーションと分析をさらに進めるために、既存のデータレイヤーを活用することもできます。

### 5. あなたのためではなく、マイクロソフトのために働くAI： アラートがない。

Clarityのようなツールの最も顕著な欠陥の1つは、プロアクティブなアラートメカニズムの欠如である。

コンバージョンを促進するためにデジタルファネルに依存している企業にとって、問題をリアルタイムで検出して対応できないことは大きなダメージになりかねません。
例えば、予約やチェックアウトプロセスの重要なステップで、特定のブラウザやデバイスのユーザーに対して問題が発生した場合を考えてみましょう。このような中断を通知する自動化されたアラートがなければ、誰も気づかないうちに数日、数週間、あるいは数ヶ月が経過してしまうかもしれません。その間、無数の潜在顧客がエラーに遭遇し、その試みを放棄し、フラストレーションから競合他社に乗り換える可能性が高い。

これは単なる憶測ではなく、私たちが業界を超えて繰り返し見てきたパターンだ。オンラインで航空券を予約しようとして、途中でエラーに遭遇することを想像してみてほしい。何度か失敗した後、代わりに別の航空会社を試してみることにするかもしれない。そうすることで、あなたはすぐに他の航空会社で購入するだけでなく、シームレスな体験を提供してくれた競合他社の生涯顧客になる可能性もあるのです。そのような顧客を失う真のコストはいくらだろうか？失われた売上が1件あたり平均数百ドルから数千ドルであり、発見されていないファネルの問題によって同様の事故が頻繁に発生する場合、累積損失は時間の経過とともに5桁から6桁に達する可能性があります。

Air360 にはプロアクティブ・アラートが含まれており、導入も使用も簡単ですが、リスクコントロールが簡単になる可能性があります。

### 6. クッキーのない未来への準備

Air360 は完全なクッキーレス・ソリューションに投資し、ウェブサイト上で起きていることの実際と全体像を確実に把握し、クッキーのない未来に備えます。Air360 は、クッキーレスでありながら、ウェブサイトが顧客に可能な限り最高のエクスペリエンスを提供しているかどうかを監視するために必要なすべての機能も提供します。

一方、ClarityはCookieに依存しているため、訪問者の部分的な情報を得ることができます。ClarityはCookieなしモードに切り替える可能性を提供しますが、このモードでは洞察という点ではまったく役に立ちません。
明らかに、マイクロソフトはクッキーに依存することを奨励しているため、クッキーから利益を得ることができます。

マイクロソフトは訪問者のデータを必要としており、そのためにクッキーを必要としています。Air360 を選択することで、あなた自身の目標に役立つ将来の準備ができているソリューションに投資することを選択します。

## 結論

Clarityのようなツールは、無料でユーザーフレンドリーであるように見えるが、隠れたコストや制限があり、コンバージョン率向上の効果を妨げています。マイクロソフトは、明らかに自社の利益のためのデータ収集に重点を置いており、自社の目標とのズレを生む可能性があります。

Clarityのような「無料」ツールの魅力は、特に厳しい予算で運営されている新興企業や小規模チームにとっては魅力的かもしれません。しかし、その裏には、当初の利便性を覆すような重大な制限やコストが潜んでいます。時間のかかるセットアップやメンテナンスに伴うサンクコストから、機能性の目に余るギャップまで、Clarityは有意義な成長に必要な実用的な洞察を提供するには不十分です。さらに懸念されるのは、このツールがユーザーの成功よりもプロバイダーの課題を優先していることです。歴史が何度も示しているように、このような基盤の上に構築されたツールがユーザーに長期的な価値をもたらすことはほとんどありません。Clarityのようなプラットフォームにコミットする前に、企業は隠れた出費を慎重に検討し、機能が自社の目標に合致しているかどうかを評価し、自社、あるいは自社の収益に役立つツールに投資するかどうかを検討する必要があります。

Microsoft Clarityは無料で提供されるため魅力的に見えるかもしれませんが、企業は隠れたトレードオフに注意する必要があります。対照的に、Air360 は安全でプライバシーを最優先した分析ソリューションを提供し、企業はデータを完全に管理することができます。
ユーザーの信頼を重視し、プライバシー規制の遵守を確実にしたいのであれば、Air360 を選択することは明らかです。


Air360 を使用すると、あなたのために働くように設計され、あなたの目標に準拠するように設計されたデータを得ることができます。
あなたが得る洞察は、あなたのコンバージョンを改善し、あなたのユーザー体験を最適化し、最終的にあなたのビジネスを前進させることに焦点を当てています。
