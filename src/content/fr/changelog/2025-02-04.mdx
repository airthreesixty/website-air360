  <span>Amélioration</span>

- Des vidéos tutorielles sont désormais disponibles auprès de l'OPA. Ces vidéos vous permettront de comprendre comment utiliser l'OPA pour vos analyses.
![](/images/changelog/2025-02-04/1.png)
- Fonctionnalité de ré-envoi d'invitation : Dans la page de gestion des équipes, il est maintenant possible de ré-inviter un utilisateur après l'expiration de l'invitation.
![](/images/changelog/2025-02-04/2.png)
- Lors de la création d'un nouvel événement d'erreur, le bouton Besoin d'aide affiche maintenant une section sur la façon de configurer les erreurs dans la fenêtre contextuelle.
![](/images/changelog/2025-02-04/3.png)
